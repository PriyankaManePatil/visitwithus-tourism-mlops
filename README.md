# Visit With Us - Tourism MLOps Project

This repository contains an end-to-end automated MLOps pipeline to predict whether a customer will purchase the Wellness Tourism Package. 
The workflow includes:

- Hugging Face Dataset Hosting
- Model Training & Hyperparameter Tuning
- Model Registry on Hugging Face Hub
- Streamlit Deployment on Hugging Face Spaces
- CI/CD Pipeline using GitHub Actions
